NodeAttributes

interface NodeAttributes : PropertiesSpec(source)

These attributes will be applied by default to all module nodes, unless overridden by that node's GraphvizModuleTypeSpec.

See the Graphviz docs for any restrictions, this interface will just pass-through the attribute and let Graphviz handle validation.
